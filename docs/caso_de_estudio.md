\# Caso de estudio MELFA



Este proyecto toma como base un notebook académico de cinemática de un robot industrial MELFA de 6 grados de libertad.



El objetivo es convertir ese notebook en un proyecto organizado y extensible usando:



\* Git,

\* GitHub,

\* modularización,

\* ambientes virtuales,

\* pruebas básicas,

\* y trabajo asistido con IA.



El notebook original contiene:



\* transformaciones homogéneas,

\* visualización 3D,

\* cinemática directa,

\* cinemática inversa,

\* ejemplos numéricos.



En este repositorio el objetivo no es solo conservar el notebook, sino empezar a separar código reusable y construir una estructura más mantenible.
